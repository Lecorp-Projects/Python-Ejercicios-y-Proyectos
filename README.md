# Proyectos Simples en Python

Este repositorio reúne distintos ejercicios, prácticas y mini proyectos desarrollados en Python durante mi proceso de aprendizaje.

La idea principal es conservar y organizar programas hechos en diferentes momentos, desde ejercicios básicos hasta proyectos un poco más completos, para poder observar mi avance, mejorar el código con el tiempo y tener un registro de lo trabajado.

## Contenido del repositorio

Aquí se pueden encontrar programas relacionados con temas como:

- Fundamentos de programación en Python.
- Uso de variables, condicionales y ciclos.
- Funciones.
- Manejo básico de cadenas de texto.
- Ejercicios de lógica.
- Talleres y prácticas realizadas durante el aprendizaje.
- Mini proyectos aplicando distintos conceptos.

## Organización

El repositorio está dividido en carpetas según el tipo de ejercicio o proyecto.  
Cada programa busca incluir comentarios iniciales que expliquen brevemente:

- Qué hace el código.
- Qué funcionalidades contiene.
- Qué conceptos de programación se practican.

## Propósito

Este repositorio tiene como objetivo:

- Guardar de forma ordenada mis ejercicios y proyectos en Python.
- Documentar mi proceso de aprendizaje.
- Practicar buenas costumbres al escribir y organizar código.
- Mejorar progresivamente programas anteriores.

## Nota

Algunos programas fueron desarrollados bajo restricciones específicas de aprendizaje, por ejemplo, evitando el uso de listas u otras estructuras de datos para practicar soluciones alternativas con los recursos disponibles en ese momento.
